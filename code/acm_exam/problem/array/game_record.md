游戏记录

[参考-游戏记录](https://www.nowcoder.com/discuss/789506479857418240)

知识点：字典、排序****13

输入一个数 n，其后跟 n 行人名和游戏，表示这个人玩了哪些游戏，要求按人名和游戏名出现的先后顺序输出每个人玩的游戏

示例 1：
输入：
5
Alice Dota2
Bob LOL
Alice CSGO
Bob Dota2
Charlie CSGO
输出：
Alice: CSGO, Dota2
Bob: Dota2, LOL
Charlie: CSGO

示例 2：
输入：
4
Bob Football
Alice Basketball
Alice Basketball
Alice Basketball
输出：
Bob: Football
Alice: Basketball

括号调整次数

[参考-小欧的括号操作](https://www.nowcoder.com/questionTerminal/0bcbd996fe234e25960abf0c82327dc2?channelPut=25seo)
[参考-括号序列](https://ac.nowcoder.com/acm/contest/185/E)

知识点：栈、贪心

给定 2 个只包括 '('，')'，'{'，'}'，'[', ']' 的字符串 s ，判断经过几次修改（只能从已有括号进行修改，不能插入或删除），字符串变成有效。  
有效字符串需满足：  
左括号必须用相同类型的右括号闭合。  
左括号必须以正确的顺序闭合。  
每个右括号都有一个对应的相同类型的左括号  

示例 1：
4  
({[]])  
({[]]}  
(  
)  
输出：  
1  
2  
-1
-1

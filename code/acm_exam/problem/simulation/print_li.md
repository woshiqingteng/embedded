打印里字

知识点：模拟

[原题-打印里字](https://blog.csdn.net/huanbailu/article/details/142853525)

输入描述
一个正整数 n，代表"里”字的大小。1 <= n <= 30
输出描述
输出 11*n 行，每行输出一个长度为 11*n 的，仅包含'.'和'*'的字符串，这些'*'从视觉上组成了'里'字的形状

示例 1:
输入：
1
输出：
...........
..*******..
..*..*..*..
..*******..
..*..*..*..
..*******..
.....*.....
..*******..
.....*.....
.*********.
...........
